## Ini apaan sih?

INSOL adalah aplikasi berbasis **CodeIgniter 4** + **PHP** yang hidup damai di folder ini.  
Versi seriusnya: web app normal.  
Versi jujurnya: tempat kita ngoding sambil ketawa tipis.

## Fitur Andalan

- Nama proyek auto sport: **Indonesia Salto**
- Ringan, kencang, nggak banyak gaya
- Ada test biar logic nggak salto ke jurang
- Struktur folder rapi, jadi drama berkurang

## Bekal Sebelum Ngoding

- PHP 8.x
- Composer
- Niat (opsional tapi disarankan)
